# Voucher Scanner Tool

This tool is compiled as a `.so` file for protection and optimized for Termux.

## How to use in Termux

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd <repo-name>
   ```

2. **Run the setup script:**
   This script will install all dependencies and compile the code for your specific Termux environment.
   ```bash
   bash setup.sh
   ```

3. **Run the tool:**
   After setup is complete, the `scanner.py` file will be deleted for security, and you can use the tool via `run.py`.
   ```bash
   python run.py
   ```

## Requirements
- Termux
- Internet Connection
- Admin approval via Telegram for License activation.
